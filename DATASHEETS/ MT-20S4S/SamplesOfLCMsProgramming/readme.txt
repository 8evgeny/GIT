*****************************************************************************
*****			Краткое описание				*****
*****************************************************************************

Данный архив содержит примеры программ на  C  подобном языке для демонстрации
алгоритмов работы с нашими ЖК индикаторами. Программы подробно прокомментиро-
ваны, но компилится не будут - не определены имена управляющих сигналов ЖКИ и
функция задержки времени.

Если у Вас есть предложения по улучшению содержимого данного архива или заме-
чания,  а также если Вы заметили ошибку в примерах - сообщите нам электронной
почтой по адресу - Петухов Дмитрий Владимирович <Dima@melt.com.ru>.  Спасибо.


*****************************************************************************
*****			Описание файлов.				*****
*****************************************************************************

AllText4.c	- пример программы для буквенно-цифровых ЖК индикаторов с 4-х битным режимом включения
AllText8.c	- пример программы для буквенно-цифровых ЖК индикаторов с 8-ми битным режимом включения
MT-6116.c	- пример программы для графического ЖК индикатора MT-6116 с любым буквенным индексом
MT-12232B.c	- пример программы для графического ЖК индикатора MT-12232B
MT-12232A,C,D.c	- пример программы для графических ЖК индикаторов MT-12232A, MT-12232C, MT-12232D
MT-12864.c	- пример программы для графического ЖК индикатора MT-12864 с любым буквенным индексом
