В данном каталоге расположены библиотеки, которые я успел написать.

Эти библиотеки не будут работать без файлов "ядра" находящихся в каталоге STM32Lib/Device

Список того, что есть:

Colors.h            - Коды цветов для графических библиотек
/Fonts              - Стандартные Adafruit шрифты
FSMC16_TFT.h        - Драйвер для ILI9341 подключенный по FSMC 16 бит
FSMC_RA8875.h       - Драйвер для дисплея на RA8875 подключение по FSMC. До конца ещё не доделан
functions.h
gfxfont.h
GFX_Button.h        - Графические кнопки для дисплеев
GFX_TFT.h           - AdafruitGFX адаптированный для STM
HX8347_8bit.h       - Драйвер для дисплея HX8347. До конца не дописан
ILI9341_8bit.h      - Драйвер для дисплея ILI9341 8 бит подключение
OneWire.h           - Драйвер 1-Wire
Print.h             - Класс Print как в ардуине
RingFIFO.h          - FIFO для UART
RS485.h             - Библиотека для передачи по протоколу RS485
SButton.h           - Библиотека Гайвера для кнопок.
SDallas.h           - Библиотека для работы с DS18B20
STimer.h            - Библиотека Гайвера для программного таймера
